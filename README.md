**Perform Landing Page Clone**

A pixel-perfect recreation of the [Perform Framer Template](https://framer.com/projects/Perform-copy--lnwMEAG7h4uMnmDdQzw0-8y8xc). This project focuses on high-end typography and minimalist UI design using only pure HTML and CSS.

**Live Demo**
[View the Live Site Here]( https://emanfatimahh31.github.io/perform-clone/)

**Design Goal**
The objective was to manually code a sophisticated design originally built in Framer to master:
Precise Positioning: Matching the original spacing and layout.
Responsive Styling: Ensuring the luxury aesthetic holds up on mobile and desktop.
Typography: Implementing clean, modern font pairings.

**Technologies Used**
HTML5: Semantic structure.
CSS3: Flexbox/Grid for layout and custom animations.
